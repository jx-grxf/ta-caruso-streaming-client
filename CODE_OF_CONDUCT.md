# Code of Conduct

## Our standard

This project aims to be respectful, constructive, and technically honest.

Examples of expected behavior:

- be respectful and patient
- give actionable feedback
- assume good intent
- disagree with ideas without attacking people
- keep discussions focused on improving the project

Examples of unacceptable behavior:

- harassment or personal attacks
- insulting, demeaning, or exclusionary language
- doxxing or sharing private information
- bad-faith trolling or intentionally disruptive behavior

## Enforcement

The maintainer may remove, edit, or reject comments, issues, commits, code, wiki edits, or other contributions that do not align with this Code of Conduct.

## Scope

This applies within project spaces such as:

- issues
- pull requests
- discussions
- linked contribution conversations
