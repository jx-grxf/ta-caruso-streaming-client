# Security Policy

## Supported scope

This project is intended for trusted local-network use.

Please report security issues related to:

- accidental exposure of local services or sensitive data
- unsafe URL fetching or SSRF-style behavior
- local file exposure
- credentials or token handling
- Electron or web UI security concerns

## Reporting

Please do **not** open a public issue for an unpatched security vulnerability.

Instead, contact the maintainer privately first:

- GitHub: [@jx-grxf](https://github.com/jx-grxf)

If private contact options expand later, this file should be updated.

## Expectations

When reporting a vulnerability, please include:

- affected version or commit
- reproduction steps
- expected impact
- any suggested mitigation if you have one

Best-effort responses are intended, but this is still an independent personal project and not a commercial support channel.
